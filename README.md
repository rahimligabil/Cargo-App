A real-time cargo tracking application designed with a modern full-stack architecture and deployed in a production environment for real-world usage.
The system provides live GPS tracking, real-time communication via WebSocket, cloud-based storage using Firebase, and is fully containerized with Docker to ensure scalability, reliability, and maintainability.


The goal of this project is to provide a scalable, reliable, and real-time cargo tracking solution where users can monitor cargo locations and status updates instantly.

The system:
Streams live GPS data using WebSocket
Synchronizes real-time updates via Firebase
Stores files and media securely using Firebase Storage
Persists application data in PostgreSQL
Follows MVC architecture for clean separation of concerns
Is fully Dockerized and deployed to a production server
